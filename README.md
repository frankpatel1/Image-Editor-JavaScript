# 🖼️ Image Editor JavaScript

A lightweight and interactive image editor built using **HTML**, **CSS**, and **JavaScript**. This project enables users to upload images and apply various editing operations directly within the browser without requiring external software.

## 📖 Overview

Image Editor JavaScript is a web-based application designed to demonstrate modern front-end development techniques and image processing functionality using native JavaScript. The application provides an intuitive interface for editing images in real time.

## ✨ Features

* Upload images from local storage
* Real-time image preview
* Apply image filters and effects
* Adjust brightness, contrast, and saturation
* Rotate and flip images
* Reset edits instantly
* Download edited images
* Responsive design for desktop and mobile devices
* Fast and lightweight implementation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Canvas API

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Image-Editor-JavaScript.git
```

Navigate to the project folder:

```bash
cd Image-Editor-JavaScript
```

Open the application:

```bash
Open index.html in your browser
```

## ▶️ How to Use

1. Launch the application.
2. Upload an image from your device.
3. Apply filters and adjustments.
4. Rotate, flip, or modify the image as needed.
5. Preview changes in real time.
6. Save the edited image to your device.
